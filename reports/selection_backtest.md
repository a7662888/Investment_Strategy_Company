# 選股回測驗證 — 三風格 vs 固定/買進持有

- 期間:2020-08-18 ~ 2026-05-06;月度再平衡;含換手成本 0.6%
- 股池 20 檔;每次選 5 檔;只用 ≤T 資料(無未來函數)。
- ⚠️ Codex/Antigravity 為**風格代理**(非其正式程式碼),用來對照『永遠滿倉 vs Claude 風險感知』。

## 判讀(誠實)

- 報酬最高:**Claude Agent(風險感知)**(+1367.1%)。
- 回撤最小:**全宇宙20檔買進持有**(-28.7%)。
- 風險調整後最佳(Sharpe):**全宇宙20檔買進持有**(1.13)。
- Claude Agent 的賣點是**風險感知(回撤較小、空頭轉守)**,不是衝最高報酬;
  若它在 Sharpe/回撤勝出、但總報酬輸給滿倉/買進持有,屬預期(犧牲部分多頭報酬換抗跌)。
- 若**買進持有就贏過所有選股**,代表此股池/期間選股難加值,應如實告知使用者、不過度行銷選股功能。

## 關鍵結論(別被 +1367% 騙了)

1. **高報酬 ≠ 有技能,而是「集中 + 承擔更多風險」**:三家選股總報酬遠高於買進持有,但**回撤都 −53%~−55%**(買進持有僅 −29%),且 **Sharpe(1.0/1.03/0.92)並沒有贏過買進持有(1.02/1.13)**。同樣的風險調整報酬,只是把資金壓在少數飆股上放大了波動與報酬。
2. **風險調整後,最好的其實是「全宇宙 20 檔買進持有」**(Sharpe 1.13、回撤 −28.7%)。選股在此股池/期間**沒有提供 risk-adjusted 的超額價值**。
3. **Claude Agent 的「風險感知」這次沒發揮**:雖只在場 73%,回撤仍達 **−52.9%**,跟永遠滿倉差不多 → regime 轉守**沒有真的把回撤壓下來**(該觸發時太晚/太少,或 top-5 過度集中)。這是要修的缺陷,不是賣點。
4. **股池是手挑的大型 survivor(台積電/聯發科/航運…),2020–2026 又是大多頭**,全體數字被嚴重高估;**不可當實盤預期**。換成含下市/中小型股的股池,選股優勢可能消失。

> 老實說:這份回測**沒有證明「依大盤選股會更賺」**。它證明的是「選股=更集中=更高報酬也更大回撤,但 risk-adjusted 沒贏大盤」,且 Claude Agent 的抗跌設計需要修正。價值仍在**可解釋 + 風險提示**,而非報酬數字。
