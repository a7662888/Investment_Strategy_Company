## 誠實判讀(結論)

1. **風險感知有效,且這就是 Claude Agent 的差異化**:同樣選股,加上 regime 曝險縮放 + 移動停損後,
   **最大回撤 −61.8% → −19.1%(砍掉約 ⅔),Sharpe 1.49 → 1.71**。代價是總報酬從 +1426% 降到 +619%
   ——**用一部分多頭報酬換到全場最小的回撤**,符合「風險感知」的定位。
2. **Claude v2 是全場 risk-adjusted 最佳**:Sharpe 1.71(高於買進持有 1.35),且**回撤最小(−19.1%)**、
   只在場 62%(空頭/高波動轉現金)。這才是「Claude 自己的 Agent」值得存在的理由——不是衝報酬,是抗跌。
3. **誠實看待絕對報酬**:每日模擬下,滿倉型策略真實回撤達 −58%~−62%(先前月度取樣低估為 −53%)。
   股池是手挑大型 survivor + 2020–2026 大多頭,**所有絕對報酬都被高估,不可當實盤預期**;
   有意義的是**相對**指標(v2 回撤與 Sharpe 全場最佳)。
4. **已知取捨**:移動停損觸發後到下次月度再平衡才回場,可能錯過急彈(whipsaw);−19% 仍是真實回撤、非無風險。

   即線上 Claude Agent 會明確告訴使用者「建議幾成倉 + 停損設多少」,與本回測一致。
