# 校準預測模型證據報告 — `claude_logit_calibrated_v1`

> Claude lane 交付。把 Codex 原本「未校準」的技術因子模型,升級為**滾動樣本外校準 + 可解釋**的版本。
> 目的:讓互動訓練的每個建議都「說得出理由、拿得出依據」,且**不對外誇大**。

## 模型

- 型態:logistic regression(11 個可解釋技術因子)。
- 標籤:**Triple-barrier**(López de Prado)—— 波動縮放的獲利門檻 / 停損門檻 + 5 日時間門檻;
  上界 = 進場價 ×(1 + 1.0×日波動×√H),下界對稱;先觸上界=1、先觸下界=0、時間到看期末報酬符號。
  比舊版「5 日漲跌」更濾雜訊、更貼交易實務(特徵只用截止日以前資料,`future_knowledge_used=false`)。
- 訓練資料:**20 檔跨產業真實台股**,2020-01 起,共 28,332 筆樣本。
- 驗證:**滾動 walk-forward** —— 每年度區段只用「該段以前」資料訓練、預測該段(真樣本外),
  池化 3 段 OOS;production 模型再以全部資料擬合(最即時)。
- 上線評分器 `company/model/score.py` 為**純標準函式庫**,Render `app.py` 可直接 import,零新增相依。

## 樣本外指標(誠實版,triple-barrier 標籤)

| 指標 | 值 | 解讀 |
|---|---|---|
| 池化 OOS AUC | 0.513 | 僅略高於隨機 0.5 — 純技術面預測本來就很難,別當水晶球 |
| 各 fold OOS AUC | 0.49 / 0.53 / 0.52 | 跨年度穩定但偏弱,沒有哪年突然神準(代表沒過擬合) |
| OOS 準確率 | 51.8% | 對比基準 52.3% |
| **IC(資訊係數)** | **0.021** | 預測機率 vs 前向報酬的日橫斷面 rank 相關 → **幾乎沒有選股排序能力** |
| **ICIR** | **0.068** | IC 穩定度極低 |
| **Precision@Top3** | **52.0%** | 每日機率最高 3 檔的實際上漲率 ≈ 基準 52.3% → **無法靠它挑「今天最強的股」** |

## 關鍵結論:模型會「估信心」,但不會「排名次」

加了 triple-barrier + IC/Precision@TopK 後,真相更清楚了:
- **IC 0.02、Precision@Top3 52%(≈基準)→ 橫斷面選股能力幾乎是零**。模型**沒辦法可靠地說「今天這 3 檔最強」**。
- **但校準桶單調**(47.8% → 58.9%,高桶 +6.6pp,平均報酬也單調上升)→ 模型**很會判斷「自己這次有多大把握」**。
- 所以模型的正確用法 = **信心過濾器**(只在高機率桶才較進取),**不是選股排名器**。
  這也解釋了為什麼選股回測要靠**風控(曝險縮放+停損)**而非「挑對股」才賺 —— 因為挑股的 edge ≈ 0。

> ⚠️ 誠實提醒:這些都是 survivor 股池 + 大多頭期的數字,**絕對值高估**;且未計交易成本。
> IC/Precision@TopK 偏弱是**真實結論**,不是 bug —— 純技術面個股日頻預測本就接近隨機。

## 每筆建議附的「理由」

`score.py` 回傳每個因子的標準化貢獻(weight × z-score),依影響力排序,例如:
`站上/跌破 20 日均線偏空(貢獻 -2.46)`、`60 日均線偏多(+2.03)`。前端可直接列出前 3–4 條作為人話理由。

## 誠實聲明 / 邊界

- AUC ~0.51、IC ~0.02 代表**單一技術模型不足以當買賣依據、也不會選股**;它的價值在「校準後的信心」與「可解釋理由」,
  用於**風險意識與部位調整**,非投資建議。高信心桶的邊際(~6.6pp)真實但不大,且未計交易成本。
- 極端桶(60%+ 僅 253 筆)樣本仍少,謹慎看待。
