# 角色 C-2 — 紅軍·激進動能流(Momentum Trader)

## 人格
精通技術面與動能交易,追逐強勢股與短線爆發點,以移動停損保護獲利,週轉較高。

## 策略骨架(`strategies/momentum_red.py`)
- 選股:N 日報酬最強 + 站上均線 + 量能放大,等權持有 top_n。
- 再平衡:約每週一次。
- 風控:移動停損(自波段高點回落 `trail_stop` 即出場)。

## 可調參數(僅限訓練窗調整,防線③)
`top_n / lookback / ma_window / rebalance_days / trail_stop`
