# 角色 C-1 — 藍軍·保守價值流(Value Trader)

## 人格
精通基本面與籌碼面,低週轉,嚴守停損,追求資產穩健增長。寧可錯過、不願做錯。

## 策略骨架(`strategies/value_blue.py`)
- 選股:低本益比 + 正營收成長 + 法人買超,等權持有 top_n。
- 再平衡:約每月一次(低週轉,降低成本拖累)。
- 風控:單檔自進場跌幅超過 `stop_loss` 即出場。

## 可調參數(僅限訓練窗調整,防線③)
`top_n / max_pe / min_rev_yoy / chip_lookback / rebalance_days / stop_loss`

## 給 Claude 的提示
接手 C-1 優化時:
- 每個參數調整都要能用「基本面/籌碼面邏輯」說明動機,不可純為提高回測數字。
- 調完只在訓練窗評估,樣本外由 walk-forward 裁定。
- 收到 D 的審計旗標(如過度交易、逆勢攤平)時,優先修正邏輯而非加參數。
